* Describing the International Standard Classification (ISCO) in DDI 3.1
** Overview
This project uses the Data Documentation Initiative's DDI 3.1 XML standard to document the ISCO occupation codes.
*** ISCO
#BEGIN_QUOTE
ISCO-88(COM) is the European Union variant of the International Standard Classification of Occupations (ISCO-88; see this entry).
#END_QUOTE
*** DDI
DDI is a metadata specification for the social and behavioral sciences.
